# 关于本项目

本项目**是** 官方的 [osu!wiki](https://osu.ppy.sh/help/wiki/Welcome) 的本地化重建版本。但是本wiki并不是由官方负责维护，在爱发电的前提下。除中文以外，其他语言会尽力保持与官方wiki的一致。
本项目中所有文章均由多位玩家或翻译组成员尽心尽力翻译成，虽然可能翻译有误或不够详细。但是我们必须**尊重**他们所付出的劳动力成果！
